# Top_frontend

## Install

- Install npm globally
  - From [website](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)
  ```bash
  npm install -g npm
  ```
- Install Node.js
  - Recomended version - 14.8 LTS
  - From [website](https://nodejs.org/en/)
  - From [nvm](https://github.com/nvm-sh/nvm) (recomended)
- Install dependencies
  ```bash
  cd Top_frontend-app
  npm install
  ```
- Run server
  ```bash
  npm run dev
  ```
- Run lint for js
  ```bash
  npm run lint
  ```
- Fix lint js errors
  ```bash
  npm run lint:fix
  ```
- Run stylelint for css
  ```bash
  npm run stylelint
  ```
- Fix css errors
  ```bash
  npm run stylelint:fix
  ```
